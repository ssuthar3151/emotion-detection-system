# Emotion Detection System

## Setup Instructions

1. **Clone the Repository**:  
   Use the following command to clone the repository to your local machine:
   ```bash
   git clone https://github.com/ssuthar3151/emotion-detection-system.git
   ```

2. **Navigate to the Project Directory**:  
   Change the current working directory to the project folder:
   ```bash
   cd emotion-detection-system
   ```

3. **Install Dependencies**:  
   Make sure you have Python installed, then create a virtual environment and install the required packages:
   ```bash
   python -m venv venv
   source venv/bin/activate   # On Windows use: venv\Scripts\activate
   pip install -r requirements.txt
   ```

4. **Run the Application**:  
   Execute the following command to start the emotion detection system:
   ```bash
   python app.py
   ```

## Features

- **Real-time Emotion Detection**: This system can detect emotions in real-time using a camera.
- **Multimodal Approach**: Combines facial expression recognition and voice tone analysis to improve accuracy.
- **User-Friendly Interface**: Provides an intuitive UI for users to interact with the application.
- **Data Visualization**: Displays emotion analytics and insights based on the detected emotions.
- **Cross-Platform Compatibility**: Works seamlessly on Windows, macOS, and Linux.
